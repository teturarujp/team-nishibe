# ディレクトリ構成について
ディレクトリ構成は大まかに以下のようになっています。

コードを確認する際にご活用ください。
```

root
├── public
│   ├── font - ttfファイル置き場
│   ├── img  - ページで使う画像置き場
│   └── favicon.ico
├── src
│   ├── components - ページで使うパーツ（ヘッダー、フッター）
│   ├── pages      - ページ本体
│   ├── styles     - css置き場
│   ├── utils      - ts置き場
│   └── main.tsx
└── README.me - 現在地

```
